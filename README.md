# HR Analytics Dashboard

## Overview
Interactive Power BI dashboard analyzing 5,000 employees across attrition, training impact, and compensation.

## Pages
1. **Executive Summary** - Key metrics and overview
2. **Attrition Analysis** - 967 employees departed (19.34% rate)
3. **Training Impact** - Relationship between training hours and attrition
4. **Compensation Analysis** - Salary distribution by department and role

## Key Findings
- HR department shows highest attrition (needs systemic review)
- Modest inverse relationship between training and attrition (confounded by tenure)
- IT and Finance departments have highest salary budgets
- Lower-paying roles correlate with higher attrition

## Data
- Dataset: 5,000 employees
- Columns: Age, Attrition, Department, Salary, Training Hours, Performance Rating, etc.

## Tools
- Power BI Desktop
- MySQL (data source)
- DAX (calculations)

## How to Use
1. Download `HR_Analysis_Dashboard.pbix`
2. Open in Power BI Desktop
3. Explore interactive dashboards
